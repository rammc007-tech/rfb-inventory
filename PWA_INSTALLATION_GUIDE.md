# 📱 PWA Installation Guide - RFB Inventory

## App Installation செய்வது எப்படி?

உங்கள் RFB Inventory app-ஐ mobile மற்றும் desktop-ல் install செய்யலாம்!

---

## 📱 Android Phone-ல் Install

### Method 1: Install Button (எளிதான வழி)
1. App-ஐ browser-ல் open செய்யுங்கள்
2. Login செய்யுங்கள்
3. Top-right corner-ல் **"Install"** button தெரியும்
4. Click செய்யுங்கள்
5. "Install" confirm செய்யுங்கள்
6. ✅ Done! Home screen-ல் app icon தெரியும்

### Method 2: Browser Menu
1. Chrome browser-ல் app open செய்யுங்கள்
2. Menu (⋮) click செய்யுங்கள்
3. **"Add to Home screen"** அல்லது **"Install app"** select செய்யுங்கள்
4. Name confirm செய்து "Add" click செய்யுங்கள்
5. ✅ Done!

---

## 🍎 iPhone/iPad-ல் Install

### Safari Browser Use செய்யுங்கள்:
1. Safari-ல் app open செய்யுங்கள்: `rfb-inventory2.vercel.app`
2. **Share button** (⬆️) tap செய்யுங்கள் (bottom-ல் middle)
3. Scroll down செய்து **"Add to Home Screen"** select செய்யுங்கள்
4. Name edit செய்து (optional) **"Add"** tap செய்யுங்கள்
5. ✅ Done! Home screen-ல் app icon தெரியும்

**Note**: iOS-ல் Chrome/Firefox-ல் install செய்ய முடியாது. Safari மட்டுமே support செய்யும்.

---

## 💻 Desktop-ல் Install (Windows/Mac/Linux)

### Chrome Browser:
1. App-ஐ Chrome-ல் open செய்யுங்கள்
2. Address bar-ல் **install icon** (⊕) தெரியும்
3. Click செய்யுங்கள்
4. "Install" confirm செய்யுங்கள்
5. ✅ Done! Desktop-ல் app shortcut create ஆகும்

### Edge Browser:
1. App-ஐ Edge-ல் open செய்யுங்கள்
2. Address bar-ல் **"App available"** notification தெரியும்
3. Click செய்து "Install" select செய்யுங்கள்
4. ✅ Done!

### Firefox:
1. Address bar-ல் **home icon with +** தெரியும்
2. Click செய்து install செய்யுங்கள்

## 🔄 App Update செய்வது எப்படி?

App automatically update ஆகும்:
1. New version deploy ஆனால்
2. App open செய்யும்போது
3. Background-ல் update download ஆகும்
4. Next time open செய்யும்போது new version load ஆகும்

Manual update:
1. App-ஐ close செய்யுங்கள்
2. மீண்டும் open செய்யுங்கள்
3. அல்லது browser cache clear செய்யுங்கள்

## 🗑️ App Uninstall செய்வது எப்படி?

### Android:
1. App icon-ஐ long press செய்யுங்கள்
2. "Uninstall" அல்லது "Remove" select செய்யுங்கள்
3. Confirm செய்யுங்கள்

### iOS:
1. App icon-ஐ long press செய்யுங்கள்
2. "Remove App" select செய்யுங்கள்
3. "Delete App" confirm செய்யுங்கள்

### Desktop:
1. App-ஐ open செய்யுங்கள்
2. Menu (⋮) → "Uninstall" select செய்யுங்கள்
3. அல்லது OS settings-ல் போய் uninstall செய்யுங்கள்

## 🆘 Troubleshooting

### Install Button தெரியவில்லை?
